A React + Vite frontend application for managing Books, Chapters, Pages, and Page Versions with Restore functionality.

Features

-Create Books (title, description, username)

-Create Chapters under selected Books

-Create Pages under selected Chapters

-Display Pages in a table

-Restore older Pages

-Automatically create new versions on restore

-Show Version history below the Pages table

-Dynamic UI updates (no refresh)

-Single global CSS file

Tech Stack

-React (Hooks)

-Vite

-Axios

-JavaScript

-HTML & CSS
